Sistema de controle de eventos desenvolvido em Grails pelo JUGMS (http://jugms.com.br)

Funcionalidades:

* Submissão de palestras e aprovação de palestras (em desenvolvimento)
* Gerenciamento de inscrições:
  - confirmação de presença
  - apontar qual palestra pretende ver (para planejar tamanho da sala de cada palestra)
  - checkin (com cadastro simplificado para quem não fez inscrição)
  - sorteio de brindes
* Avaliação das palestras pelas inscritos
* Envio de email para os inscritos
